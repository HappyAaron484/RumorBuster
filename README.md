# 流言終結者#22 資產配置模擬器
模擬定期定額投資（DCA）策略的 Python 程式，預設標的為 ETF（交易所交易基金）投資組合。可以根據不同的風險偏好選擇 ETF 投資組合，並查看投資回測結果。

## 功能
- 從 Yahoo Finance 獲取 ETF 歷史價格數據。
- 計算最大回撤（MDD）和年化報酬率（CAGR）。
- 模擬定期定額投資，並計算最終市值、總投入資金和總報酬率。
- 繪製 ETF 價格走勢圖。
